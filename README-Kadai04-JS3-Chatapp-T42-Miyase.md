## ①課題内容（どんな作品か）
- お互いのコメントが交互に表示され、また列毎に色分けされている。
　Firebaseを通じてGoogle認証にてログイン、利用する

## ②工夫した点・こだわった点
-　一旦大枠の形を作り、Firebaseでの認証を付加した所。
　　チャットの最新から10個のコメントのみ表示される（全体で）所。

　質問する時間がなかった為、ここは継続して改良します。
